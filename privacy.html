Privacy Policy

Last updated: 12 July 2026

1. Introduction

This Privacy Policy explains how NoxiousBot ("the Bot") collects, uses, and stores information when you interact with it in Discord.

2. Information Collected

The Bot collects the following information:

  • Discord User ID and username — used to identify you for ticket ownership, bug report attribution, and role management
  • Twitch or YouTube account details — collected during content creator registration to link your streaming accounts and assign roles
  • Bug report and suggestion content — including title, description, steps to reproduce, version info, and any attached file URLs
  • Ticket content — messages submitted through the ticket modal and subsequent thread replies

3. How Information Is Used

Collected data is used solely to provide the Bot's features:

  • Filing bug reports and suggestions as GitHub issues (your Discord username is included for attribution)
  • Creating and managing support tickets as private Discord threads
  • Assigning and managing Discord roles for content creators and live streamers
  • Syncing issue status updates between GitHub and Discord
  • AI-assisted bug triage (severity, category, and duplicate detection) via Anthropic's Claude API

4. Data Storage

User data is stored in Cloudflare Workers KV, a key-value database. Data is retained indefinitely until a user or server admin requests deletion. Ticket transcripts may be stored in a Discord channel designated by the server admin.

5. Data Shared with Third Parties

  • GitHub — bug reports and suggestions are filed as public issues on GitHub repositories. Your Discord username and ID are included in these issues.
  • Anthropic — bug report text (title, description, steps) may be sent to Anthropic's Claude API for automated triage. No personally identifying information beyond the report content is sent.
  • Discord — the Bot interacts with the Discord API to send messages, manage roles, and create threads as part of its normal operation.
  • Twitch — Twitch API is used to verify streamer registration and manage EventSub subscriptions for live notifications.

6. Data Retention

  • Discord user IDs and linked account data are retained until deletion is requested
  • Bug reports and suggestions are retained as long as the associated GitHub issue exists
  • Ticket transcripts are retained according to the server's retention settings
  • Twitch EventSub subscriptions are automatically expired after a safety window

7. Data Deletion

You may request deletion of your data by contacting the developer via Discord or email at cogito.ergo.olfacio@gmail.com. Upon request, the Bot will remove your stored data, including linked account information and ticket history.

8. Security

The Bot operates on Cloudflare's infrastructure and follows standard security practices. Secrets and API keys are stored securely and never exposed to users. The Bot does not collect or store passwords, payment information, or other sensitive personal data.

9. Children's Privacy

The Bot is not directed at children under 13. It does not knowingly collect personal information from children. Users must meet Discord's minimum age requirements to interact with the Bot.

10. Changes to This Policy

This Privacy Policy may be updated from time to time. Changes will be reflected by the "Last updated" date at the top of this page.

11. Contact

For privacy-related inquiries, contact the developer at cogito.ergo.olfacio@gmail.com or via Discord.
